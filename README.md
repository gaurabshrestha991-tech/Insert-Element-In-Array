# Insert-Element-In-Array
This C++ program inserts a new element into an array at a user-specified position.

## The program:
- Takes the size of the array as input.
- Accepts array elements from the user.
- Asks the user for the position where the new element should be inserted.
- Accepts the value to be inserted.
- Shifts the existing elements to the right.
- Inserts the new element at the specified position.
- Displays the updated array.

## Concepts Used
- Arrays
- User Input/Output
- Loops
- Conditional Statements
- Element Shifting
